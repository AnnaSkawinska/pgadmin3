Constructeur graphique de requêtes

Le constructeur graphique de requêtes (connu sous l'acronyme GQB) fait partie de l'outil de requêtage. Il vous permet de construire des requêtes simples graphiquement.

Il existe trois parties principales dans GQB. Le navigateur de relation sur la gauche vous permet de sélectionner les catalogues, classes et vues à inclure dans vos requêtes. Tout en bas se trouve un groupe d'onglets vous permettant de spécifier des critères de sélection, un format de sortie et un ordre de tri. La troisième section est la partie graphique où vous pouvez dessiner les relations entre les tables. Vous pouvez ajuster la taille relative de chaque section.

Pour ajouter des relations à la requête, soit vous cliquez deux fois dans le navigateur de relations, soit vous les traînez du navigateur vers la partie graphique. Vous pouvez arranger les relations sur la partie graphique en sélectionnant le nom de la relation avec la souris et en déplaçant la relation dans la position désirée.

Pour créer des jointures entre les relations, traînez une colonne d'une relation vers une autre. Une ligne sera dessinée entre elles pour indiquer la jointure. Un clic droit sur la jointure présentera un menu contextuel à partir duquel vous pouvez sélectionner l'opérateur de jointure ou la supprimer.

Pour sélectionner les colonnes qui apparaîtront dans les résultats de la requête, vérifiez les colonnes désirées dans la relation sur la partie graphique. Chaque colonne sélectionnée sera ajoutée sur la liste sur l'onglet Colonnes pour lequel vous pouvez ajouter l'ordre et spécifier les alias de colonne.

Pour spécifier le critère de sélection, ajoutez les lignes dans la grille de l'onglet Critères. La valeur restreinte peut être initialisée par un nom de colonne ou une valeur constante. La colonne Opérateur vous permet de sélectionner un opérateur simple à utiliser pour comparer la valeur restreinte avec la valeur spécifiée dans la colonne valeur. Le connecteur spécifie comment le critère sera joint à la suivante (si nécessaire).

Les résultats de la requête peuvent être ordonnés dans l'onglet des tris. Sélectionnez les colonnes requises sur la gauche et utilisez les boutons du milieu pour les ajouter à la liste de trus sur la droite. Vous pouvez aussi supprimer les colonnes de la même façon. Pour chaque colonne ajoutée dans la liste de tri, vous pouvez indiquer la direction. Vous pouvez aussi ajuster la priorité de chaque colonne en la sélectionnant et en la déplaçant vers le haut ou vers le bas en utilisant les boutons sur la droite.

Quand vous avez terminé la conception de votre requête, vous pouvez utiliser les options Exécuter, Exécuter dans un fichier ou Exécuter un EXPLAIN sur la barre d'outils (et le menu Requêtes) pour générer et exécuter la requête.
