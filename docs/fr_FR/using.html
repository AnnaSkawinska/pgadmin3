Utilisation de pgAdmin

Cette section explique comment vous pouvez utiliser pgAdmin pour maintenir vos bases de données PostgreSQL. pgAdmin supporte les versions 7.3 et les versions ultérieures. Les versions plus anciennes ne sont pas supportés, utilisez dans ce cas pgAdmin II.

  • La fenêtre principale affiche la structure de la base de données et tous les détails des objets qu'elle contient.

  • Utilisez le menu Add server.. ou le bouton de la barre d'outil pour ajouter une nouvelle connexion à pgAdmin.

  • Avec Modifier le mot de passe, vous pouvez changer le mot de passe de l'utilisateur actuellement connecté sur la base sélectionnée.

  • Contrôler le serveur vous permet de contrôler le statut des serveurs de bases de données, les démarrer et les arrêter.

  • L'outil de requêtage permet l'exécution de commandes SQL arbitraires.

  • Le débogueur permet le débogage de fonctions et procédures stockées PL/pgsql et EDB-spl.

  • L'outil d'export permet l'export des données provenant de l'outil de requêtage.

  • La grille d'édition permet la visualisation et l'édition de données dans une table ou une vue sélectionnée.

  • L'outil de maintenance exécute différentes tâches, comme la reconstruction des statistiques sur les bases de données et les tables, nettoie les données inutilisées et réorganise les index.

  • L'outil de sauvegarde appelle l'outil de sauvegarde PostgreSQL pour créer les fichiers de sauvegarde de vos données.

  • L'outil de restauration appelle l'outil de restauration PostgreSQL pour restaurer les données à partir de vos fichiers de sauvegarde.

  • L'assistant de gestion des droits permet l'affectation des droits aux utilisateurs/groupes sur un grand nombre d'objet en une fois.

  • L'outil de création de rapports génère des rapports simples à partir de données et d'objets de votre base de données.

  • La fenêtre d'état du serveur affichera les utilisateurs actuellement connectés, les verrous et le journal applicatif du serveur.

  • Le dialogue Options vous permet de configurer pgAdmin.

  • Les astuces de gourous vous aident à perfectionner vos bases de données.

  • Les options en ligne de commande vous permettent de sélectionner le meilleur mode pour démarrer pgAdmin.

Lors de l'édition des propriétés d'un objet de la base de données, pgAdmin vous apportera toute l'aide nécessaire sur les commandes SQL nécessaires grâce à la touche de fonction F1. Pour que cela fonctionne, l'aide SQL du site doit être configurée correctement dans le dialogue des options.
