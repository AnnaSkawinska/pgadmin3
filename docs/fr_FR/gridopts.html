Options de visualisation des données

Lorsque « Appliquer le filtre et visualiser les données » est sélectionné à partir du menu principal ou lorsqu'un clic a lieu sur le bouton « Tri / Filtre » dans le visualisateur de données, ce dialogue apparaît. Il autorise la modification de l'ordre du tri des données et l'application d'un filtre pour restreindre les données affichées.

Vous pouvez sélectionner une ou plusieurs colonnes ainsi que la direction du tri qui doivent être utilisés pour récupérer les données de la base.





Vous pouvez saisir une clause WHERE arbitraire pour restreindre l'ensemble de résultats qui est récupéré de la base de données. Vous pouvez cliquer sur « Valider » pour que pgAdminIII teste si la clause WHERE est valide.
