Assistant des droits

L'assistant des droits permet l'affectation d'un ensemble de droits aux groupes et utilisateurs à plusieurs objets (tables, séquences, vues et fonctions) d'une façon agréable.

L'onglet Sélection affichera tous les objets applicatbles sous l'objet actuellement sélectionné. Vérifiez tous les objets pour lesquels vous souhaitez configurer les droits.

L'onglet Sécurité définit les droits et les groupes/utilisateurs qui peuvent être concernés. N'affecter aucun droit à un groupe ou un utilisateur résultera en la suppression de leur droits.

L'onglet SQL affiche les commandes SQL qui peuvent être générés à partir des cibles sélectionnées.

Lors du clic sur le bouton OK, les commandes SQL seront exécutées ; tout résultat sera affiché dans l'onglet Messages.
