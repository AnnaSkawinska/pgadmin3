pgAgent

Introduit avec pgAdmin III v1.4, pgAgent est un agent de planification de jobs pour PostgreSQL, capable d'exécuter des travaux complexes en plusieurs étapes, composées de scripts shell et de scripts SQL.

À partir de pgAdmin 1.9, pgAgent est distribué indépendamment de pgAdmin. Il peut être téléchargé à partir de l' aire de télépchargement du site de pgAdmin.

  • Installation - pgAgent peut être installé sur un ou plusieurs serveurs.

  • Créer des jobs - les jobs peuvent être facilement créés en utilisant pgAdmin III.

  • Créer des planifications - chaque job peut avoir une ou plusieurs planifications.

  • Créer des étapes - chaque job consiste en une ou plusieurs étapes, chacune pouvant être un script SQL ou un script shell.
