Macros dans l'outil de requêtage

Les macros de l'outil de requêtage vous permettent d'exécuter des requêtes SQL prédéfinies avec un simple appui sur une touche. Les requêtes prédéfinies peuvent contenir des emplacement du type $SELECTION$. Lors de l'exécution de la macro, il sera remplacé avec le texte actuellement sélectionné dans le panneau SQL de l'outil de requêtage.

Pour créer une macro, sélectionner l'option « Gérer les macros » à partir du menu Macros de l'outil de requêtage. Sélectionnez la touche que vous voulez utiliser, saisissez le nom de la macro et la requête. Ajoutez en option le texte $SELECTION$, puis cliquez sur le bouton Sauvegarder pour enregistrer la macro.

Pour effacer une macro, sélectionner la macro sur le dialogue « Gérer les macros », puis cliquez sur le bouton « Effacer ».

Pour exécuter une macro, appuyez simplement sur la touche appropriée, ou sélectionnez à partir du menu Macros.
