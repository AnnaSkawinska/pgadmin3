Aperçu de Slony-I avec pgAdmin III

Les objets de Slony-I sont intégrés dans le navigateur de pgAdmin, permettant d'avoir une seule interface pour gérer les objets de la base et pour administrer la réplication.

L'onglet de statistiques pour la collection de nœuds ainsi que pour les nœuds individuels montrent le statut de la queue des événements de réplication, et permet la surveillance du bon fonctionnement de Slony.
