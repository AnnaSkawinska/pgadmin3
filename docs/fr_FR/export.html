Export de données dans pgAdmin

Les données sont écrites dans un fichier en utilisant les options sélectionnées.

L'option du séparateur de lignes sélectionne le caractère utilisé pour séparer deux lignes de données. Sur les systèmes Unix, il s'agit habituellement du caractère nouvelle ligne (LF). Sur Windows, la paire CR/LF est souvent utilisée.

Le caractère de séparation entre les colonnes peut être une virgule, un point-virgule ou une barre verticale.

Les colonnes individuelles peuvent être englobées dans des guillemets. Cela peut seulement s'appliquer aux colonnes de chaînes de caractères (les colonnes numériques n'en ont pas besoin) ou à toutes les colonnes quelque que soit leur type. Le caractère utilisé pour les guillemets est soit le guillemet simple soit le guillemet double.

Si l'option « Nom des colonnes » est sélectionnée, la première ligne du fichier contiendra le nom des colonnes.
