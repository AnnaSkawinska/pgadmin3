Slony-I Supported Platforms

Slony-I has been verified by to work on the platforms which are listed below. Slony-I can be successfully built, installed and run on these platforms.

Last updated: Jun 23, 2005

If you experience problems in these platforms, please subscribe to Slony-I Mailing List and ask your questions.
