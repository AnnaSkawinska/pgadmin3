Extended functions

pgAdmin III can display some more detailed data, if the server is instrumentized using the admin module for PostgreSQL 8.x. This will contain helper functions. The admin module is available at this place. For Windows, the installer includes the module and will install it automatically unless it is deselected.

If the functions are loaded on a server, the following functions will be available:

  • The sizes on disk of tablespaces, databases, tables and indexes are displayed on the Statistics tab in the main window.
  • The Status dialogue will have the logfile tab and can display the server logfile if it is configured.
  • The Rotation button on the Status dialogue logfile tab will not get enabled on 8.0 servers, because the needed support functions were not implemented. If you need triggering the server logfile rotation, please contact us.
